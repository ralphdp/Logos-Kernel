# Logos Boilerplate — V6.0 (Generative)

High-fidelity reference implementation for the **Logos Kernel** (v1.1.0+).

- **Architecture:** Next.js 16.1.6 (Turbopack), TypeScript.
- **Styling:** Vanilla CSS / Tailwind (Truth to Materials).
- **ORM:** Prisma / Drizzle (PostgreSQL).
- **Graphics:** Three.js (React Three Fiber / Drei).
- **Core Logic:** Recursive Sync (L2-Grammar aligned).

This is the **one high-fidelity template** — not ten messy examples. It is designed for high-resolution structural reasoning under the [Logos Protocol](https://logos.pub).

---

## Getting Started

1. **Clone the repository:**
   ```bash
   npx @ralphdp/logos-boilerplate-nextjs@latest my-logos-app
   ```

2. **Install dependencies:**
   ```bash
   cd my-logos-app
   npm install
   ```

3. **Set up database:**
   ```bash
   npm run db:push
   ```

4. **Start development:**
   ```bash
   npm run dev
   ```
